# 👿 라이어게임:비밀의 문

## ✋ 프로젝트 소개
#### 예능으로만 보던 그 게임을 실시간으로 즐기는 온라인 게임입니다.

### 🔍 주요 기능
- `화상` 실시간으로 상대방의 표정을 관찰하면서 라이어를 찾아내세요.
- `채팅` 채팅으로 참가자들과 대화를 나누며 라이어를 찾아내세요.
